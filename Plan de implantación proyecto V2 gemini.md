Contexto: Actúa como un Arquitecto de Software Senior especializado en desarrollo multiplataforma con Flutter.

Objetivo: Diseñar un Plan de Implantación detallado para el proyecto "Ferretería", una aplicación robusta que debe funcionar en Android, Web, Windows e iOS utilizando el entorno Antigravity.

Restricciones Técnicas:

Stack: Flutter & Dart.

Backend: Firebase (Firestore para base de datos y Authentication para login con correo/password).

Gestión de Estado: Provider.

IDE: Configuración específica para Antigravity / VS Code.

Estándares: No utilizar opciones de producción "A", no incluir analíticas. El enfoque debe ser un estándar limpio y escalable.

Tarea: Antes de generar cualquier línea de código, genera un Plan de Implementación en formato Markdown que incluya:

Requerimientos de Entorno: Herramientas necesarias y configuración del SDK para Antigravity.

Arquitectura de la Información (UI/UX): Estructura de vistas (Home, Inventario, Carrito, Perfil).

Configuración de Firebase: Pasos para Firestore y Auth sin analíticas.

Estructura del Proyecto: Organización de carpetas (Models, Views, Controllers/Providers, Services).

Dependencias Críticas: Listado para el pubspec.yaml (provider, cloud_firestore, firebase_auth, etc.).

Cronograma de Desarrollo (Fases): Desde la inicialización hasta el despliegue multiplataforma.

Formato de entrega: Tablas de Markdown organizadas y listas de verificación (checklists). No proporciones el código fuente todavía, solo el plan estratégico.
